# Daily-Budget-App

App to manage your Monthly/Daily Budget written in C.



- Add Income/Expense
- Input amount, category (food, rent, travel, etc.), and date.
- Daily Budget Tracking
- Compare daily spending against a set daily limit.
- Monthly Summary
- Show total income, expenses, and savings.
- Categories
- Breakdown of expenses by category (e.g., food 40%, rent 30%).
- Reports
- Text-based reports showing trends (could later be extended to CSV export).
- Persistent Storage
